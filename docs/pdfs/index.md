- [Lecture on MAC protocol1.pdf](../Lecture on MAC protocol1.pdf) — MAC layer and media access control
- [Lecture on wireless.pdf](../Lecture on wireless.pdf) — Wireless networking lecture
- [Routing Protocol.pdf](../Routing Protocol.pdf) — routing protocol slides
- [routing_protocol.pdf](../routing_protocol.pdf) — routing protocol notes or slides (different version)
- [Subnetting and VLSM.pdf](../Subnetting and VLSM.pdf) — subnetting and VLSM lecture slides
- [interface_between_process_and_network.pdf](../interface_between_process_and_network.pdf) — interface between processes and network

> Note: PDFs currently live in the repository root; if you prefer them organized under `/docs/pdfs/` I can copy or rename them and update the links.

## Recommended Usage
Open the PDFs to follow slides and examples in the routing pages:

- Use `routing/rip.md` while reading RIP sections
- Use `routing/ospf.md` while reading OSPF sections
- Use `routing/bgp.md` while reading BGP sections
